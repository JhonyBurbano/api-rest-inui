# api-rest-inui

## Descripción

This api-rest allows you to manage (GET, LIST, CREATE, UPDATE, DELETE) patient information for doctors and add notes (CREATE, LIST_NOTE_PATIENT)

## Instalación

## Install dependencies:
``$ npm i ``

## Build Api
``npm run start:dev``
